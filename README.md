# MD Viewer

A lightweight desktop markdown viewer and editor built with Electron.

## Features

- Open and render markdown files (GitHub-flavored)
- Edit mode with raw markdown editing
- Save changes (Ctrl+S / Cmd+S)
- Refresh from disk (F5)
- Light/Dark theme toggle

## Setup

```bash
npm install
npm start
```

## Build Portable Exe

```bash
npm run dist
```

Output will be in `dist/`.
